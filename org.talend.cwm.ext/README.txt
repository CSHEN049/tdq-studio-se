EMF model extending the CWM model for Talend's use. This model will be refered as 
TdCWM model hereafter. 

Short description of packages:
- org.talend.cwm.constants:  contains constants used by some CWM classes.
- org.talend.cwm.helper: Helper classes to be used when handling CWM and TdCWM classes.
- org.talend.cwm.relational.* : generated EMF packages
- org.talend.cwm.softwaredeployment.* : generated EMF packages
