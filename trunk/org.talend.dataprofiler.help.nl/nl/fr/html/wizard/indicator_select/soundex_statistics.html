Statistiques de fréquence Soundex :

  Fournit des indicateurs pour analyser la fréquence phonétique de vos données. Il est utile afin d'identifier les fautes d'orthographe ou de frappe.

  Par exemple, "abod" et "adobe" produiront le même son même si ces mots sont différents. Ces indicateurs afficheront que "adobe" a deux orthographes différentes.

  La table de fréquence soundex affiche les sons les plus fréquents dans vos données et combien de fois ils apparaissent.

  La table de basse fréquence soundex affiche les sons les moins fréquents et combien de fois ils apparaissent.
