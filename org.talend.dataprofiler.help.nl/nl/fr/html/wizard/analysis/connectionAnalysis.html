Analyse de connexion :

Vue d'ensemble de la structure de la base de données

Ce type d'analyse vous donne une vue d'ensemble du contenu de votre base de données. Il calcule le nombre de tables et le nombre de lignes par table pour chaque catalogue et/ou schéma, etc. Il calcule également le nombre de clés d'index et de clés primaires, etc.

Il vous permet de voir, en un coup d'oeil, quelle base de données contient le plus de données, quelles tables sont vides, etc.
