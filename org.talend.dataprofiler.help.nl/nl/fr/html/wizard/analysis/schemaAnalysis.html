Analyse de schéma :

Vue d'ensemble de la structure du schéma

Ce type d'analyse vous donne une vue d'ensemble du contenu d'un schéma. Il calcule le nombre de tables et le nombre de lignes par table pour chaque catalogue et/ou schéma...

Utilisez ce type d'analyse pour analyser un schéma Oracle (et d'autres SGBD). Si vous souhaitez analyser une base de données MySQL, choisissez une "Analyse de catalogue" à la place.
