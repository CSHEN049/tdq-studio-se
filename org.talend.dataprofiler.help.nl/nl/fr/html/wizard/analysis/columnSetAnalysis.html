Analyse de colonnes :

Analyse d'un ensemble de colonnes

Ce type d'analyse s'applique sur un ensemble de colonnes (pouvant définir un enregistrement complet) et non sur chaque colonne séparémment comme cela est le cas pour l'analyse de colonnes. Le résultat de cette analyse fournit les informations suivantes :
nombre de lignes
nombre d'enregistrements distincts
nombre d'enregistrements uniques
nombre de doublons

Avec le moteur Java, il est également possible d'appliquer des indicateurs de correspondance de modèles. Le résultat de l'analyse donne alors le nombre d'enregistrements vérifiant tous les modèles sélectionnés.
