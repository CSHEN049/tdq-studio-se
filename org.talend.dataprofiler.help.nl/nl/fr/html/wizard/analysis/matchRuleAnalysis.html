Analyse de similaires

Ce type d'analyse permet d'évaluer la quantité de données similaires ou en doublon. Cette analyse calcule le nombre de groupes de données similaires dans une table ou un ensemble de colonnes. Vous pouvez également l'utiliser pour définir des règles de correspondance qui peuvent ensuite être partagés dans le référentiel.
