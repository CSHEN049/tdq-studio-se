Longueur de texte :

Lorsque vous calculez la longueur de données texte, les options suivantes sont disponibles :

- compte des nulls : quand cette option est sélectionnée, les données nulles sont comptées comme des champs de texte de longueur zéro. When not selected, null data are not taken into account in the results.

- count blanks: when selected, blank texts (e.g. " ") are counted as zero length text fields. Lorsqu'elle n'est pas sélectionnée, les données nulles se sont pas prises en compte dans les résultats.

Note : Sous Oracle, les chaînes de caractères vides et nulles sont considérées comme les mêmes éléments. Vous devez donc cocher ou décocher les cases afin d'obtenir des résultats cohérents.
