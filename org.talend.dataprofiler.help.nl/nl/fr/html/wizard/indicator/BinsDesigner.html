Assistant de création des intervalles :

Dans ce panneau, vous pouvez définir les intervalles à utiliser lorsque vous devrez agréger des données continues dans des groupes.

Les groupes créés auront tous la même taille, sauf le premier et le dernier.

La valeur minimale est celle du début du premier groupe.

La valeur maximale est celle de la fin du dernier groupe.

Exemple : min = 0 max = 100 nb = 4
Les groupes sont
[0, 25[ U [25, 50[ U [50, 75[ U [75, 100[
Lorsque le nombre de groupes défini est 0, aucun paquet n'est créé : L'indicateur calcule la fréquence de chaque valeur de la colonne.
