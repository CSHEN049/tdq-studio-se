Options des caractères d'espace :

Lorsque vous agrégez des données texte, les options suivantes sont disponibles :

- agréger les nulls et les blancs : quand cette option est sélectionnée, les données nulles sont considérées comme des champs de texte de longueur zéro. Cela signifie qu'elles sont traitées comme une chaîne de caractères. Lorsque cette option n'est pas sélectionnée, les données nulles sont traitées comme toutes les autres données texte.

- agréger les blancs : quand cette option est sélectionnée, les textes blancs (p. ex. " ") sont tous regroupés et considérés comme des chaînes de caractères vides. Lorsque cette option n'est pas sélectionnée, les textes blancs sont traités comme n'importe quelles autres données texte.

Note : Sous Oracle, les chaînes de caractères vides ou nulles sont considérées comme étant les mêmes éléments. Vous devez donc cocher ou décocher ici les cases afin d'obtenir des résultats cohérents.
