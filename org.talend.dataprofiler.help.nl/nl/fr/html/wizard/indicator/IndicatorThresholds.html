Seuils sur les indicateurs :

Ici, vous pouvez définir les seuils attendus sur la valeur de l'indicateur.

La plage définie ici est utilisée pour mesurer la qualité des données.

Si la valeur de l'indicateur est en-dehors de la plage définie, cela signifie que les données sont de mauvaise qualité.

Par exemple, si vous définissez une plage [0, 3] sur l'indicateur de compte blanc, cela signifie que vous souhaitez que le nombre de champs blancs soit inférieur à 3 (inclus).

Vous pouvez définir un seul seuil ou n'en définir aucun. Vous devez définir ces seuils soit par valeur soit par pourcentage en respectant le nombre de lignes.
