Statistiques avancées :

  Fournit d'autres indicateurs pour une analyse plus détaillée de vos données.

  La table de fréquence affiche les valeurs les plus fréquentes et combien de fois elles apparaissent.

  La table de basse fréquence affiche les valeurs les moins fréquentes et combien de fois elles apparaissent. Il est utile afin de détecter les problèmes de qualité de données.

  Le mode est la valeur la plus probable, la valeur apparaissant le plus souvent dans vos données.
