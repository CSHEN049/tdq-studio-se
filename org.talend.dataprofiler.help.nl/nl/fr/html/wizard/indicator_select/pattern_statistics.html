Statistiques de fréquence des modèles :

  Fournit des indicateurs pour analyser la structure de vos données. Remplace tous les caractères alphabétiques en minuscules par "a", tous les caractères alphabétiques en majuscules par "A" et tous les caractères numériques par "9".

  Par exemple, un code postal anglais tel que "SL6 1NJ" donnera le modèle suivant : "AA9 9AA".

  La table de fréquence des modèles affiche les modèles les plus fréquents dans vos données et le nombre de fois qu'ils apparaissent.

  La table de basse fréquence des modèles affiche les modèles les moins fréquents et le nombre de fois qu'ils apparaissent.

  La table de fréquence de modèles de date affiche les modèles fréquents ayant un format de date spécialement conçu dans vos données et combien de fois ils apparaissent. Ne l'exécutez que dans le moteur Java.
