Statistiques des numéros de téléphone :

  Propose des indicateurs pour analyser les numéros de téléphone.

  Par exemple, un indicateur va compter le nombre de numéros de téléphone valides, un autre va compter le nombre de numéros de téléphone ayant un code région valide.

  Plusieurs formats existent : Le format national pour lequel il n'y a pas de préfixe de pays, le format international avec un préfixe de pays, le format E.164 ayant au maximum quinze chiffres et généralement écrit avec un préfixe +.

  Notez que ces indicateurs sont uniquement disponibles lorsque vous sélectionnez le moteur Java dans les paramètres de l'analyse.
