Column Correlation Analysis:

Analyse de corrélation numérique

Ce type d'analyse affiche les corrélations entre les données nominales et numériques dans un graphique à bulles. Pour chaque valeur nominale, la moyenne des données numériques est calculée et représentée sur l'axe vertical, et la valeur nominale est représentée sur l'axe horizontal.

L'un des objectifs de cette analyse est d'identifier facilement les problèmes de qualité de données en regardant les valeurs extrêmes dans le graphique.
