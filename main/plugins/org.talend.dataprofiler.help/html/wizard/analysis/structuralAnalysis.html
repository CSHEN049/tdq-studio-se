Structural Analysis:

The analyses available in this folder return an overview of the content of your database. They compute the number of tables and the number of rows per table for each catalog and/or schema, etc. They also count the number of indexes and primary keys, etc.

This will help you to see at a glance which database contains the most data, which tables are empty and so on.
