Phone Number statistics:

  Offers indicators to analyze the phone numbers.

  For example, an indicator will count the number of valid phone numbers, another will count the number of phone numbers that have a valid region code.

  Several formats exist: National format for which there is no country prefix. International format with a country prefix, E.164 format which can have a maximum of fifteen digits and are usually written with a + prefix.

  Note that these indicators are only available when you select the Java engine in the analysis settings.
