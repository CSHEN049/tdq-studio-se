Column Correlation Analysis:
-------------------------------------
Numerical Correlation Analysis
This kind of analysis shows correlations between nominal and numerical data in a bubble chart. For each nominal value, the average of the numerical data is computed and represented on the vertical axis.
One of the aim of this analysis is to easily find data quality issues by looking at the extremal values in the graph.
Time Correlation Analysis
This kind of analysis shows correlations between nominal and time data in a kind of gantt chart. For each nominal value, the minimal and maximal values of the time data is computed and represented as a bar in the chart.
Nominal Correlation Analysis
This kind of analysis works with only nominal values and shows the relations between pairs of data in network graph. It highlights weak relationships between pairs of data in order to find potential incorrect relationships.
