Schema Analysis:
-------------------------------------
Schema Structure Overview
This kind of analysis gives you an overview of the content of a schema. It computes the number of tables and the number of rows per table for each catalog and/or schema...
Use it to analyze an Oracle (and some other DBMS) schema. If you want to analyze a database on MySQL, choose a "Catalog analysis" instead.
