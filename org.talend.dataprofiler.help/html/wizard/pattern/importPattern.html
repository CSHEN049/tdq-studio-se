Import Pattern:
--------------------------------------

You can import several regular expressions from an xls file.

The file format is the following:
  • C1: Label: the label of the pattern (must not be empty)
  • C2: Purpose: the purpose of the pattern (can be empty)
  • C3: Description: the description of the pattern (can be empty)
  • C4: Regular expression: the regular expression applicable to all databases (can be empty)
  • C5: MySQL Regexp: the regular expression applicable to MySQL databases (can be empty)
  • C6: Oracle Regexp: the regular expression applicable to Oracle databases (can be empty)
  • C7: Author: the author of the regular expression (can be empty)
