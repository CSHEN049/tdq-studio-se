Import Pattern:
--------------------------------------

You can import several regular expressions from an csv file.

The file format is the following:
  • C1: Label: the label of the pattern (must not be empty)
  • C2: Purpose: the purpose of the pattern (can be empty)
  • C3: Description: the description of the pattern (can be empty)
  • C4: Author: the author of the regular expression (can be empty)
  • C5: Relative Path: the relative path to the root folder (can be empty)
  • C6: All DB Regular: the regular expression applicable to all databases (can be empty)
  • C7: DB2 Regexp: the regular expression applicable to DB2 databases (can be empty)
  • C8: MySQL Regexp: the regular expression applicable to MySQL databases (can be empty)
  • C9: Oracle Regexp: the regular expression applicable to Oracle databases (can be empty)
  • C10: PostgreSQL Regexp: the regular expression applicable to PostgreSQL databases (can be empty)
  • C11: SQL Server Regexp: the regular expression applicable to SQL Server databases (can be empty)
