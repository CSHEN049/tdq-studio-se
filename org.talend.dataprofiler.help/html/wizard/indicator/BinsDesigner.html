Bins designer:
-------------------------------------
In this panel, you can define the bins to use when you must aggregate continuous data into discrete bins.

The created bins will all have the same size, except the first and last bins.

The minimal value is the beginning of the first bin.

The maximal value is the end of the last bin.

Example: min = 0 max = 100 nb = 4
The bins are
[0, 25[ U [25, 50[ U [50, 75[ U [75, 100[
When the number of bins is set to 0, no bin is created: The indicator computes the frequency of each value of the column.
