Text Length:
-------------------------------------

When computing the length of a text data, the following options are available:

- count nulls: when selected, null data is counted as a zero length text field. When not selected, null data are not taken into account in the results.

- count blanks: when selected, blank texts (e.g. " ") are counted as zero length text fields. When not selected, blank texts are not taken into account in the results.
